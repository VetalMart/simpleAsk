Ціль коротко: автоматично формувать журнал заявок.

Алгоритм роботи оператора:
1. скачати необхідні файли в папку;
2. запустити программу;
3. задати номер останньої заявки;
4. передивитись інформацію в прототипі журналу;
5. скопіювати інформацію із прототипу в оригінал журналу;
6. передивитись файли в папці "неформат"
7. роздрукувати файли із папки "формат"

Алгортим роботи программи: 
1. відкрити папку із файлами;
2. перевірити ці файли на відповідність;
3. підходящі файли оброблюєм, не підходящі переміщаєм в папку 
    "неформат";
4. взяти із підходящих файлів інформацію;
5. вкласти цю інформацію в прототип журнал із присвоїним унікальним 
    номером та датою;
6. присвоїти цим файлам цей номер і дату;
7. зберегти ці файли в папці "формат";

Що потрібно для запуску:
1. встановити Python за ссилкою:
    https://www.python.org/downloads/
